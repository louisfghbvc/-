專案2 - Kai Liu Going Down OpenCV版
---
 * 遊戲介紹 
 * - 簡單的下樓梯(凱留版)，新增一些不同的樓梯，每50層新增不同樓梯，第100層開啟凱留加速
 * 遊戲介面 
 * - 選擇單人或雙人
 * - 選擇難易度(小心服用，建議玩簡單)
 * 遊戲玩法
 * - 使用綠色物件控制玩家移動方向
 * 遊戲道具 
 * - 灰色 - (一般)
 * - 綠色 - (補血)
 * - 紅色 - (減血)
 * - 黃色 - (碰到直接消除)
 * - 紫色 - 左或右(正向加速，反向減速)
 * - 50層  黑色 - 同紫色，1秒後消除
 * - 150層 藍色 - 一般、較快的樓梯
 * - 200層 白色 - 往上移動100距離，並消除
